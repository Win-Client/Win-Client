# 📖 Win-Client — Usage & Installation Guide

Get up and running with **Win-Client** on your local machine. Follow this clean guide to configure your environment and launch the client successfully.

---

## 🛠️ Required Dependencies

Potion is built on the modern **Fabric** toolchain. Ensure you have the following requirements installed in your Minecraft client profile:

1.  **Fabric Loader**: Set up the latest stable loader version for your desired Minecraft version.
2.  **Fabric API**: Handles critical hooks and event systems.
    *   👉 **Download:** [Fabric API on Modrinth](https://modrinth.com/mod/fabric-api)
3.  **Fabric Language Kotlin**: Enables seamless execution of Kotlin-based client mechanics.
    *   👉 **Download:** [Fabric Language Kotlin on Modrinth](https://modrinth.com/mod/fabric-language-kotlin)
4. **Minecraft 26.1.2**
---

## ⚙️ Quick Installation

1.  Download or build `WinClient.jar`.
2.  Navigate to your active Minecraft directory:
    *   **Windows:** `%appdata%\.minecraft\mods`
    *   **macOS:** `~/Library/Application Support/minecraft/mods`
    *   **Linux:** `~/.minecraft/mods`
3.  Place **WinClient.jar**, **Fabric API**, and **Fabric Language Kotlin** into the `mods` folder.
4.  Launch the game using your configured Fabric Loader profile.

---

## 🎮 Interface Controls

Once the client is active in-game, control all modules via the clean HUD dashboard:

*   ⌨️ **Open ClickGUI:** Press `Right Shift` (RSHIFT) to trigger the interactive visual control panel.
*   🖱️ **Module Interaction:** Right-click modules within the ClickGUI window to expand their precise configuration sliders, toggles, and binding fields.
